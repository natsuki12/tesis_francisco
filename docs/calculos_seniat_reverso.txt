╔══════════════════════════════════════════════════════════════════════════════╗
║  ESTRUCTURA GENERAL DEL REVERSO                                            ║
╚══════════════════════════════════════════════════════════════════════════════╝

  El Reverso contiene la Sección J - ANEXOS, que es el desglose detallado
  de todos los activos, pasivos y exclusiones que alimentan los cálculos
  del Anverso (Sección I).

  Encabezado: Repite las secciones A-D del Anverso (Datos del Contribuyente,
  Datos del Causante, Dirección, Representante Legal).

  Subsecciones del Reverso:
    1. Bienes Inmuebles
    2. Bienes Muebles
    3. Pasivos
    4. Desgravámenes
    5. Exenciones
    6. Exoneraciones
    7. Bienes Litigiosos

  Cada subsección tiene:
    • Una tabla con columnas específicas
    • Una fila final "Monto Total" con la sumatoria


╔══════════════════════════════════════════════════════════════════════════════╗
║  SUBSECCIÓN 1: BIENES INMUEBLES                                            ║
╚══════════════════════════════════════════════════════════════════════════════╝

  Columnas:
  ┌──────────┬──────────────────┬──────────────────┬───────────┬──────────┬──────────────┐
  │ Tipo     │ Descripción      │ Registro         │ Vivienda  │ Bien     │ Monto        │
  │          │                  │                  │ Principal │ Litigioso│ Declarado    │
  └──────────┴──────────────────┴──────────────────┴───────────┴──────────┴──────────────┘

  Datos del ejemplo:
  ┌──────────┬──────────────────────────────────────┬─────────┬──────────┬──────────────┐
  │ Townhouse│ 100% de casa quinta tipo townhouse    │ (datos  │   SÍ    │    NO        │ 3,700,000.00
  │          │ identificada con el numero 04-04.     │  de     │ (verde) │   (rojo)     │
  │          │ Tipo de Bien Inmueble: Townhouse.     │ registro│         │              │
  │          │ Linderos: norte con townhouse 04-03;  │ público)│         │              │
  │          │ sur con townhouse 04-05; este con     │         │         │              │
  │          │ zona de jardín... Superficie          │         │         │              │
  │          │ Construida: 184 metros cuadrados.     │         │         │              │
  │          │ Dirección: Cunjunto residencial       │         │         │              │
  │          │ "Las tunas", urbanización Maneiro.    │         │         │              │
  └──────────┴──────────────────────────────────────┴─────────┴──────────┴──────────────┘
                                                              Monto Total: 3,700,000.00

  Campos especiales:
    • Vivienda Principal: Botón verde SÍ / rojo NO
      → Si es SÍ, el monto se replica automáticamente en Desgravámenes
    • Bien Litigioso: Botón verde SÍ / rojo NO
      → Si es SÍ, el monto se muestra en la tabla de Bienes Litigiosos

  Campos de la Descripción (contenido expandido):
    • Tipo de Bien Inmueble
    • Linderos (norte, sur, este, oeste)
    • Superficie Construida (m²)
    • Superficie Sin Construir
    • Área o Superficie
    • Dirección del inmueble

  Campos del Registro:
    • Oficina Subalterna/Juzgado/Notaría/Misión Vivienda
    • Número de Registro
    • Libro, Tomo, Protocolo
    • Fecha de Registro
    • Trimestre
    • Asiento Registral
    • Matrícula
    • Libro de Folio Real del Año

  Cálculo:
    Monto Total = Σ de Monto Declarado de cada bien inmueble
    → Este valor alimenta el Ítem 1 del Anverso (Total Bienes Inmuebles)

  │ Otros                      │ 0.01% de texto que esta en la descrip.  │ NO │   100.00 │

  Tipos de Bienes Muebles posibles:
    • Acciones
    • Plantaciones
    • Caja Ahorro (Caja de Ahorro)
    • Otros
    • Opciones de Compra / Opciones de Compras

  Campos de la Descripción según tipo:
    • Porcentaje de participación (%)
    • Nombre de la Empresa
    • RIF de la Empresa
    • Nombre del Oferente (para Opciones de Compra)

  Cálculo:
    Monto Total = Σ de Monto Declarado de cada bien mueble
    → Este valor alimenta el Ítem 2 del Anverso (Total Bienes Muebles)

  Tipos de Pasivos posibles:
    • Deuda Otro Especifique
    • Deuda Tarjeta de Crédito
    • Gasto Apertura de Testamento
    • Deuda Préstamos, Cuentas y Efectos por Pagar
    • Deuda Crédito Hipotecario

  Campos de la Descripción según tipo:
    • Descripción libre
    • Banco (nombre del banco)
    • Número de Cuenta/Tarjeta

  Cálculo:
    Monto Total = Σ de Monto Declarado de cada pasivo
    → Este valor alimenta el Ítem 10 del Anverso (Total Pasivo)

  NOTA: Los pasivos incluyen tanto DEUDAS como GASTOS.
  En el menú lateral del sistema están separados en:
    • "Pasivos Deuda" (deudas del causante)
    • "Pasivos Gastos" (gastos funerarios, testamento, etc.)
  Pero en el Reverso se agrupan todos bajo "Pasivos".

╔══════════════════════════════════════════════════════════════════════════════╗
║  OBSERVACIONES DEL REVERSO                                                 ║
╚══════════════════════════════════════════════════════════════════════════════╝

  1. VIVIENDA PRINCIPAL → DESGRAVAMEN AUTOMÁTICO:
     Cuando un inmueble se marca como "Vivienda Principal = SÍ", su
     monto se replica automáticamente como un desgravamen. Esto explica
     por qué Bienes Inmuebles y Desgravámenes tienen el mismo monto
     (3,700,000.00) en este ejemplo.

  2. BIEN LITIGIOSO → TABLA SEPARADA:
     Los bienes marcados como litigiosos se excluyen de la sección I
     del anverso y se muestran en la tabla de Bienes Litigiosos del
     reverso. En el ejemplo no hay bienes litigiosos.

  3. PASIVOS AGRUPADOS:
     El menú lateral separa "Pasivos Deuda" y "Pasivos Gastos", pero
     en el reverso se agrupan todos bajo una sola tabla "Pasivos".
     Los tipos observados incluyen tanto deudas (tarjeta, hipoteca,
     préstamos) como gastos (apertura de testamento).

  4. ENCABEZADO REPETIDO:
     El reverso repite las secciones A-D del anverso (contribuyente,
     causante, dirección, representante) antes de los Anexos.

  5. CÁLCULOS DEL REVERSO:
     Los ÚNICOS cálculos en el reverso son las SUMATORIAS de cada
     subsección (fila "Monto Total"). No hay fórmulas complejas.
     Toda la lógica de impuesto está en el Anverso.

  6. FORMATO DE MONTOS:
     Todos los montos usan formato con separador de miles (punto)
     y 2 decimales: ej. 3,700,000.00 o 3.700.000,00

  7. INDICADORES VISUALES:
     • Vivienda Principal: Botón circular verde (SÍ) / rojo (NO)
     • Bien Litigioso: Botón circular verde (SÍ) / rojo (NO)
     Estos botones NO son clicables en la vista de declaración
     (son de solo lectura).
