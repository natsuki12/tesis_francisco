# Reglas de Implementación

## 1. Aprobación antes de implementar
Antes de realizar cualquier cambio en el código, debo:
- Explicar los cambios propuestos
- Esperar la aprobación explícita del usuario
- Solo proceder después de recibir confirmación

## 2. Eliminación de archivos - Confirmación doble
Para eliminar cualquier archivo se requiere **confirmación doble**:
1. Primera solicitud: Pedir permiso para eliminar el archivo
2. Esperar respuesta afirmativa del usuario
3. Segunda solicitud: Confirmar nuevamente antes de proceder
4. Esperar segunda respuesta afirmativa del usuario
5. Solo entonces proceder con la eliminación

**IMPORTANTE**: No eliminar ningún archivo sin haber recibido DOS confirmaciones explícitas del usuario.
